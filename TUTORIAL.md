First dl, and compile the deps with:
```
$ make dep
```

then you can go into the directory `_drw` with:
```
$ cd _drw/
```

compile the small helper module for lists with:
```
$ make -f drw.mk
```

and then test the program with:
```
$ make -f drw.mk test
```

now you can have the list of the commands, with:
```
usage
```
